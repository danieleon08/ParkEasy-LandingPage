# **App Name**: ParkEasy Landing

## Core Features:

- Header/Navbar Component: Displays the ParkEasy logo, navigation menu (Inicio, Funcionalidades, Descarga, Contacto), and a prominent 'Descargar App' (Download App) button.
- Hero Section Component: Features an image or animation representing mobility or parking, an impactful main title (e.g., 'Encuentra parqueaderos f√°cilmente con ParkEasy' - Find parking easily with ParkEasy), a short subtitle, and a download button.
- Problem Section Component: Briefly explains the problem of traffic and lack of parking, using icons or illustrations.
- Solution Section Component: Provides a brief description of how the app works, highlighting key points such as real-time search, reviews, reservations, and favorites.
- Features Section Component: Presents main functionalities through cards with icons, titles, and descriptions. Includes functionality for admins.
- Download Section Component: Showcases an image of a phone displaying the app, download buttons for the App Store and Google Play, and a clear call to action.
- Footer Component: Includes social media links, contact email, and copyright information.

## Style Guidelines:

- Primary color: Light blue (#27dddf) to convey innovation and mobility.
- Secondary color: Yellow (#ffa300) for highlights and calls to action.
- Accent color: Dark blue (#0054b3) for a professional and trustworthy feel.
- Headline font: 'Poppins' (sans-serif) for a modern and legible style. Note: currently only Google Fonts are supported.
- Body font: 'Roboto' (sans-serif) for text and paragraphs ensuring readability. Note: currently only Google Fonts are supported.
- Use a consistent set of modern icons related to parking, mobility, and city life.
- Implement subtle fade-in and slide-up animations on scroll to create a smooth and engaging user experience.
